# К столу! — база знаний и документация продукта

«К столу!» — приложение для планирования мероприятий: подбор рецептов, расчёт ингредиентов и формирование списка покупок под гостей и меню.

## Зачем проект
- Упростить подготовку к мероприятию: меню → ингредиенты → агрегированный список покупок
- Дать удобный контроль: статусы покупок, замены ингредиентов, перерасчёт порций

## Моя роль
Роль: **системный аналитик** (pet-project в паре с наставником).  
Моя зона ответственности: требования и артефакты — **_Vision & Scope_**, **_Use Case_**, **_User Stories_**, **_SRS_**, модели данных (**_ER/UML_**), диаграммы процессов.
